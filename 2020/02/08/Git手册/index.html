git pull 时经常遇到 error: cannot lock ref 'refs/remotes/origin/dev': is at xxx but expected xxx

问题原因：

  • 在git push的时候使用了git push –force，导致远端分支被覆盖，导致你本地的refs与远端不一致。
  • 删除掉这个远端分支又重新新建了这个分支也会出现同样的问题。
