Java Crash 默认处理

CrashHandler 处理 Java 异常流程：

  • 区分Debug模式和Release模式、主进程和子进程、主线程和子线程来处理
  • 捕获Activity的生命周期内异常，并主动杀死Activity
  • View绘制流程异常捕获
  • 自定义上报
